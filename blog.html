Unity团队协作PlasticSCM的使用2023-06-01

当你需要制作一个大型项目时，必须和你的团队一起制作，这是PlasticSCM就帮上大忙了！

首先你们团队需要每个人都将UnityHub升级到最新版，并且统一你们使用的unity版本

然后团队中要有一个人去创建团队，并且将大家都拉入团队

这样你们在制作项目的时候，只需要把自己的成果点击上传

进过简单的审核之后，你的队员就能在云端项目里面看到你制作的内容并且添加到自己的电脑里面 十分的方便！

利用blender将sp导出贴图绑定fbx导出为glb格式2023-06-01

当我们用maya制作了一个简单的模型之后，我们通常会将它导入进sp中进行贴图，但是这时会遇到一个问题，就是当你贴图之后，你的贴图导入其他软件之时，软件并不会自动吸附，这就导致我们不得不自己亲手一个个对应他们，造成了很多不必要的麻烦，不过有些软件也有相应的插件解决相关的问题，这里要介绍的就是利用blender来解决

你需要将你的模型导入到blender中，然后点击你的模型，在右边找到贴图渲染，此时你会看到blender的可视化编辑器

此时只需要利用快捷键CTRL+SHIFT+T就可以唤出贴图导入页面

然后只需要选择你需要的贴图，就可以了！

最后一步非常简单就是导出你们的模型，选择glb格式，这时你会发现你的模型导入到其他的软件中贴图都不会丢失

Unity使用C#连接数据库出现中文乱码现象 2023-06-01

首先确定自己的数据库字符集设置是否正确，数据库在没有正确设置的情况下会无法写入中文，具体的解决方式就是在创建数据库表的时候设置

可以在创建表时加入下面的sql语句

ENGINE=InnoDB DEFAULT CHARSET=utf8;

如果这样还是没有解决问题可以检查自己在C#中的代码，只要在最后面加上一个Charset=utf8就行了

string conString = $"Database={databaseName};port={port};Data source={host};Charset=utf8; User Id={userName};" + $"Password={password}"; mySqlConnection = new MySqlConnection(conString); mySqlConnection.Open();
